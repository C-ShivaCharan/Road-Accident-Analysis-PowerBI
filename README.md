# Road Accident Analysis Using Power BI

## Project Overview
This project analyzes road accident data to identify patterns, trends, and key factors contributing to accidents.

## Tools Used
- Power BI
- Excel
- Data Visualization
- DAX

## Key Insights
- Total number of accidents
- Total casualties
- Accident severity distribution
- Vehicle type analysis
- Road type accident analysis
